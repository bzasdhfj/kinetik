## 📌 正文内容（支持一键复制）

一直想找一个干净、好看且不耍流氓的 Mac 打卡软件，但市面上的要么是臃肿的 Web 套壳应用（动辄几百兆内存），要么就是要强制联网注册、甚至偷偷收集隐私……

索性自己动手，用纯原生 SwiftUI 撸了一个自律打卡神器 —— **Kinetik**！✨

### 💡 为什么它与众不同？
1. 🧩 **超高颜值桌面小组件**：把 GitHub 风格的 contribution 贡献网格直接放在 Mac 桌面上！不需要打开 App，在桌面点击小组件就能一键打卡，极其丝滑。
2. 🎯 **任务主次分明**：区分“核心习惯（Required）”与“加分任务（Bonus）”，再也不会因为偶尔没完成非核心任务而感到焦虑了。
3. 🌙 **修仙党福音 (3 AM 跨天逻辑)**：经常熬夜到凌晨？别担心！Kinetik 逻辑跨天时间定在凌晨 3 点，过了零点打卡依然算在今天，Streak（连胜）绝不断档！
4. 🔒 **纯本地、零侵入**：没有任何账号、服务器或统计后台。数据 100% 存在你 Mac 本地，启动只占 20MB 内存，秒开无延迟。
